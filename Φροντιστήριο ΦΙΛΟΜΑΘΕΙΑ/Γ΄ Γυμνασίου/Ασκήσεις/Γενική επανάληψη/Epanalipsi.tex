\textbf{{\large ΛΥΜΕΝΟ ΠΑΡΑΔΕΙΓΜΑ - ΤΑΥΤΟΤΗΤΕΣ}}\\
Να βρεθούν τα αναπτύγματα από τις ταυτότητες.
\begin{multicols}{5}
	\begin{enumerate}[label=\roman*.]
		\item $ \left(x+2 \right)^2 $
		\item $ \left(4x-3 \right)^2 $
		\item $ \left(x+3 \right)^3 $
		\item $ \left(2y-5 \right)^3 $
		\item $ (x+4)(x-4) $
	\end{enumerate}
\end{multicols}
\noindent
\textbf{{\large ΛΥΣΗ}}\\
Για να βρεθεί το κάθε ανάπτυγμα το μόνο που χρειάζεται είναι να γνωρίζουμε τους τύπους σε κάθε περίπτωση. Οπότε έχουμε :
\begin{enumerate}[label=\roman*.]
\item $ \left(x+2 \right)^2=x^2+2\cdot x\cdot2+2^2=x^2+4x+4 $
\item $ \left(4x-3 \right)^2=(4x)^2-2\cdot4x\cdot3+3^2=16x^2-24x+9 $
\item $ \left(x+3 \right)^3=x^3+3\cdot x^2\cdot3+3\cdot x\cdot3^2+3^3=x^3+9x^2+27x+27 $
\item $ \left(2y-5 \right)^3=(2y)^3-3\cdot(2y)^2\cdot5+3\cdot2y\cdot5^2-5^3=8y^3-60y^2+150y-125 $
\item $ (x+4)(x-4)=x^2-4^2=x^2-16 $
\end{enumerate}
\paragraph{\large ΑΣΚΗΣΕΙΣ}
\begin{enumerate}[itemsep=5mm]
	\item Να βρεθούν τα αναπτύγματα από τις παρακάτω ταυτότητες :

\item $ (3x+2y)^2 $

\end{enumerate}

\end{enumerate}

\textbf{{\large ΛΥΜΕΝΟ ΠΑΡΑΔΕΙΓΜΑ - ΠΑΡΑΓΟΝΤΟΠΟΙΗΣΗ}}\\
Να παραγοντοποιηθούν οι παρακάτω παραστάσεις.
\begin{multicols}{2}

\item $ 3x^2-9x $
\item $ 2y^2-4y+xy-2x $
\item $ x^2-16 $
\item $ x^2+2x+1 $
\end{enumerate}
\end{multicols}
\noindent
\textbf{{\large ΛΥΣΗ}}
\begin{enumerate}[label=\roman*.,itemsep=0mm]
	\item Η παράσταση αυτή παραγοντοποιείται με τον 1ο τρόπο δηλαδή βγάζοντας κοινό παράγοντα. Ο κοινός παράγοντας θα έχει συντελεστή το Μ.Κ.Δ. των συντελεστών των προσθετέων και κύριο μέρος, τις κοινές μεταβλητές στη μικρότερη δύναμη. Διαιρώντας κάθε όρο με τον κοινό παράγοντα προκύπτουν οι όροι μέσα στην παρένθεση \[ 3x^2-9x=3x(x-3) \]
\item Στην περίπτωση που δεν υπάρχει κοινός παράγοντας σε όλους τους όρους τότε τους μοιράζουμε σε κατάλληλες ομάδες ώστε από κάθε ομάδα να μπορέσουμε να βγάλουμε κάτι κοινό.
\[ 2\tikzmark{a}y^2-4\tikzmark{b}y+x\tikzmark{c}y-2\tikzmark{d}x\tikz[overlay,remember picture]
{\draw[latex-latex,square arrow] (a.south) to (b.south);} \tikz[overlay,remember picture]
{\draw[latex-latex,square arrow] (c.south) to (d.south);}=2y(y-2)+x(y-2)\]
Στη συνέχεια θα προκύψει μια κοινή παρένθεση η οποία είναι κοινό παράγοντας άρα θα έχουμε \[ 2y(y-2)+x(y-2)=(y-2)(2y+x) \]
\item Σε κάθε ταυτότητα όπου το ανάπτυγμά της πρέπει να γραφτεί με τη μορφή γινομένου έχουμε μια μορφή παραγοντοποίησης. Στη στγκεκριμένη περίπτωση στη διαφορά τετραγώνων θα πρέπει να δούμε ποιοί ήταν οι αριθμοί που υψώθηκαν στο τετράγωνο και να μετατραπεί σε γινόμενο αθροίσματος επί διαφοράς \[ x^2-16=x^2-4^2=(x-4)(x+4) \]
\item Η παράσταση αυτή παριστάνει ανάπτυγμα ταυτότητας άρα θα γραφτεί στην προηγούμενη μορφή \[ x^2+2x+1=x^2+2\cdot x\cdot1+1^2=(x+1)^2 \]
\end{enumerate}

\textbf{{\large ΛΥΜΕΝΟ ΠΑΡΑΔΕΙΓΜΑ - Ε.Κ.Π. - Μ.Κ.Δ.}}\\
Να βρεθεί το Ε.Κ.Π. και ο Μ.Κ.Δ. από τις παρακάτω αργεβρικές παραστάσεις.
\begin{multicols}{2}
	\begin{enumerate}[label=\roman*.]
		\item $ 12x^3y^4z\;,\;16x^4y^2x^5\;,\;18x^3y^3 $
		\item $ 3x^3+9x^2\;,\;x^2-9\;,\;x^2-6x+9 $
	\end{enumerate}
\end{multicols}
\noindent
\textbf{{\large ΛΥΣΗ}}\\
Οι δύο κανόνες για την εύρεση Ε.Κ.Π. και Μ.Κ.Δ. είναι 
\begin{enumerate}[label=\arabic*.]
	\item Ε.Κ.Π. : Όλοι οι παράγοντες ο καθένας στη μεγαλύτερη δύναμη.
	\item Μ.Κ.Δ. : Μόνο οι κοινοί παράγοντες ο καθένας στη μικρότερη δύναμη.
\end{enumerate}
Αν θέλουμε να βρούμε Ε.Κ.Π. και Μ.Κ.Δ. από πολυώνυμα, πρώτα τα παραγοντοποιούμε.
\begin{enumerate}[label=\roman*.]
	\item Τα μονώνυμα δε χρειάζονται παραγοντοποίηση. Θα υπολογίσουμε Ε.Κ.Π. και Μ.Κ.Δ. πρώτα από τους αριθμούς.
	\[ E.K.\varPi.(12,16,18)=144 \]
Οπότε θα έχουμε \[ E.K.\varPi.=144x^4y^4z^5 \]
Για το Μ.Κ.Δ. θα έχουμε $ M.K.\varDelta.(12,16,18)=2 $. ¶ρα \[ M.K.\varDelta.=2x^3y^2 \]
\newpage
\item Τα πολυώνυμα πρώτα θα παραγοντοποιηθούν.
\begin{enumerate}[label=\alph*.]
	\item $ 3x^3+9x^2=3x^2(x+3) $
	\item $ x^2-9=(x-3)(x+3) $
	\item $ x^2-6x+9=(x+3)^2 $
\end{enumerate}
¶ρα θα έχουμε $ E.K.\varPi.(3,1,1)=3 $ οπότε
\[ E.K.\varPi.=3x^2(x-3)(x+3)^2 \]
Επίσης $ Μ.Κ.\varDelta.(3,1,1)=1 $ άρα \[ M.K.\varDelta.=x-3 \]
\end{enumerate}
\paragraph{\large ΑΣΚΗΣΕΙΣ}
\begin{enumerate}
	\item Να βρεθούν το Ε.Κ.Π. και ο Μ.Κ.Δ. των παρακάτω μονωνύμων
\begin{enumerate}[label=\roman*.]
	\item $ 4x^3y^2\;,\;12x^4y^3z\;,\;8xy^3z^3 $
	\item $ 9a^4\beta^5\gamma^2\;,\;27a^3\gamma^2\;,\;18a^4 $
\end{enumerate}
\item Να βρεθούν το Ε.Κ.Π. και ο Μ.Κ.Δ. των παρακάτω πολυωνύμων
\begin{enumerate}[label=\roman*.]
\item $ 4x^2-8x\;,\;3x^3-27x\;,\;4x^2-24x+36 $
\item $ x^3-2x^2+x\;,\;x-x^2\ $
	\end{enumerate}
\end{enumerate}
 \textbf{{\large ΛΥΜΕΝΟ ΠΑΡΑΔΕΙΓΜΑ - ΡΗΤΕΣ ΠΑΡΑΣΤΑΣΕΙΣ}}\\
Να βρεθούν οι τιμές τις μεταβλητής $ x $ ώστε να ορίζεται η παρακάτω παράσταση. Στη συνέχεια να απλοποιηθεί. \[ \frac{2x-4}{x^2-4} \]
\textbf{{\large ΛΥΣΗ}}\\
Για να ορίζεται μια ρητή παράσταση θα πρέπει ο παρονομαστής της να είναι διάφορος του 0. Οπότε
\[ \textrm{Πρέπει }x^2-4\neq0\Rightarrow x^2\neq4\Rightarrow x\neq\pm2 \]
Η παράσταση λοιπόν ορίζεται για όλες τις τιμές εκτός από το $ -2 $ και το $2$.\\
Για να απλοποιηθεί μια ρητή παράσταση θα πρέπει να έχουμε και στον αριθμητή και στον παρονομαστή μόνο γινόμενο. Παραγοντοποιούμε και τους δυο όρους και ύστερα διαιρούμε.
\[ \frac{2x-4}{x^2-4}= \frac{2(x-2)}{(x-2)(x+2)}=\frac{2}{x+2} \] 
\textbf{{\large ΑΣΚΗΣΕΙΣ}}
\begin{enumerate}
\item Να βρεθούν οι τιμές τις μεταβλητής $ x $ ώστε να ορίζονται οι παρακάτω παραστάσεις. Στη συνέχεια να απλοποιηθούν.
\begin{multicols}{3}
\begin{enumerate}[label=\roman*.]
\item $ \dfrac{x^2-x}{(x-1)^2} $
\item $ \dfrac{3-x}{x^2-3x} $
\item $ \dfrac{2x^2+2x}{x^3+2x^2+x} $
\end{enumerate}
\end{multicols}
\end{enumerate}

\textbf{{\large ΛΥΜΕΝΟ ΠΑΡΑΔΕΙΓΜΑ - ΕΞΙΣΩΣΕΙΣ 1ου ΒΑΘΜΟΥ}}\\
Να λυθεί η παρακάτω εξίσωση \[ \dfrac{x-1}{2}+\dfrac{x}{3}=1 \]
\textbf{{\large ΛΥΣΗ}}\\
Η διαδικασία για να λυθεί μια εξίσωση 1ου βαθμού είναι η εξής
\begin{enumerate}[label=\roman*.]
	\item Βρίσκουμε το Ε.Κ.Π. των παρονομαστών (αν υπάρχουν).
	\item Πολλαπλασιάζουμε όλους τους όρους της εξίσωσης με το Ε.Κ.Π.
	\item Απλοποιούμε το Ε.Κ.Π. με κάθε παρονομαστή. (Απαλοιφή παρονομαστών)
	\item Απαλοίφουμε τις παρενθέσεις (αν προκύψουν).
	\item Χωρίζουμε τους γνωστούς από τους άγνωστους όρους.
	\item Αφου προστεθούν οι όμοιοι όροι διαιρούμε με το συντελεστή του αγνώστου και βρίσκουμε τη λύση.
\begin{gather*}
\dfrac{x-1}{2}+\dfrac{x}{3}=1\Rightarrow\\
6\cdot\dfrac{x-1}{2}+6\cdot\dfrac{x}{3}=6\cdot1 \Rightarrow\\
3(x-1)+2x=6\Rightarrow\\
3x-3+2x=6\Rightarrow\\
3x+2x=6+3\Rightarrow\\
5x=9\Rightarrow x=\frac{9}{5}
\end{gather*}
\end{enumerate}
