\section{Η έννοια της μεταβλητής - Αλγεβρικές παραστάσεις}
\orismoi
\Orismos{Μεταβλητή} Μεταβλητή ονομάζεται το γράμμα που χρησιμοποιούμε για να συμβολίσουμε οποιονδήποτε άγνωστο αριθμό. Ο συνηθέστερος συμβολισμός για έναν άγνωστο αριθμό είναι το γράμμα $ x $.\\\\
\Orismos{Αριθμητική παράσταση} Αριθμητική ονομάζεται μια παράσταση η οποία περιέχει πράξεις με αριθμούς.\\\\
\Orismos{Αλγεβρική παράσταση} Αλγεβρική ονομάζεται η παράσταση η οποία περιέχει πράξεις με αριθμούς αλλα και μεταβλητές. \begin{itemize}[itemsep=0mm]
\item Οι προσθετέοι μιας αλγεβρικής παράστασης ονομάζονται \textbf{όροι} της.
\item Τιμή μιας αλγεβρικής παράστασης ονομάζεται ο αριθμός που προκύπτει ύστερα από πράξεις εαν αντικαταστήσουμε της μεταβλητές με αριθμούς.
\end{itemize}
\Orismos{Αναγωγή ομοίων όρων} Αναγωγή ομοίων όρων ονομάζεται η διαδικασία με την οποία γράφουμε σε απλούστερη μορφή μια αλγεβρική παράσταση προσθέτοντας μεταξύ τους τους όμοιους όρους.
\methodologia
\Methodologia{Τιμή αλγεβρικήσ παράστασησ} Για να υπολογίσουμε την τιμή μιας αλγεβρικής παράστασης για συγκεκριμένες τιμές των μεταβλητών της
\begin{enumerate}[itemsep=0mm]
\item Αντικαθιστούμε τις μεταβλητές με τις τιμές που μας δίνονται.
\item Εκτελούμε τις πράξεις με τη σωστή σειρά.
\end{enumerate}
\Methodologia{Αναγωγή ομοίων όρων} Προκειμένου να απλοποιήσουμε μια αλγεβρική παράσταση χρησιμοποιώντας αναγωγή ομοίων όρων θα πρέπει να τη φέρουμε σε τέτοια μορφή ώστε να μπορούν να προστεθούν οι όροι αυτοί.\\
Εαν δηλαδή η παράσταση περιέχει παρενθέσεις ή κλάσματα θα πρέπει με κατάλληλες μεθόδους που θα συνατήσουμε παρακάτω, τα απαλοιφθούν ώστε να μπορέσουμε να προσθέσουμε τους όμοιους όρους της.

\section{Εξισώσεισ 1\textsuperscript{\MakeLowercase{ου}} βαθμού}
\orismoi
\Orismos{Εξίσωση} Εξίσωση ονομάζεται κάθε ισότητα που περιέχει μεταβλητές.
\begin{itemize}[itemsep=0mm]
\item Εξίσωση \textbf{με εναν άγνωστο} ονομάζεται η εξίσωση που περιέχει μια μεταβλητή.
\item \textbf{Λύση} μιας εξίσωσης ονομάζεται ο αριθμός που την επαληθεύει.
\item Αν μια εξίσωση έχει λύσεις όλους τους αριθμούς ονομάζεται \textbf{αόριστη ή ταυτότητα}.
\item Αν μια εξίσωση δεν έχει καθόλου λύσεις ονομάζεται \textbf{αδύνατη}.
\item H παράσταση που βρίσκεται αριστερά του $ = $ ονομάζεται \textbf{1\textsuperscript{ο} μέλος} ενώ αυτή που βρίσκεται στα δεξιά \textbf{2\textsuperscript{ο} μέλος}.
\item \textbf{Άγνωστοι} ονομάζονται οι όροι της εξίσωσης οι οποίοι περιέχουν τη μεταβλητή,
ενώ \textbf{γνωστοί} ονομάζονται οι αριθμοί δηλαδή οι σταθεροί όροι της εξίσωσης.
\end{itemize}
\Orismos{ΕΠΑΛΗΘΕΥΣΗ}
Επαλήθευση ονομάζεται η διαδικασία με την οποία εξετάζουμε αν ένας αριθμός είναι
λύση μιας εξίσωσης, αντικαθιστώντας τη μεταβλητή της εξίσωσης με τον αριθμό αυτό.\\\\
\Orismos{Απαλοιφή παρονομαστών} Απαλοιφή παρονομαστών ονομάζεται η διαδικασία με την οποία διώχνουμε τους παρονομαστές που βρίσκονται μέσα σε μια ισότητα ή εξίσωση.\\\\
\thewrhmata
\Thewrhma{Ιδιότητεσ Ισοτήτων}
Σε κάθε ισότητα μπορούμε να προσθέσουμε, να αφαιρέσουμε, να πολλαπλασιάσουμε και να διαιρέσουμε τα δύο μέλη της με τον ίδιο αριθμό.
\[ a=\beta\Rightarrow \ccases{a+\gamma=\beta+\gamma\\a-\gamma=\beta-\gamma\\a\cdot\gamma=\beta\cdot\gamma\\\dfrac{a}{\gamma}=\dfrac{\beta}{\gamma}\;\;\;,\;\;\;\gamma\neq0} \]\mbox{}\\
\methodologia
\Methodologia{Επίλυση εξίσωσησ}
Για την επίλυση μιας εξίσωσης ακολουθούμε την παρακάτω διαδικασία : 
\begin{enumerate}[itemsep=0mm]
\item Χωρίζουμε μέλη τους γνωστούς από τους άγνωστους όρους της εξίσωσης αλλάζοντας το πρόσημό τους.
\item Κάνουμε αναγωγή ομοίων όρων και στα δύο μέλη της εξίσωσης.
\item Διαιρούμε και τα δύο μέλη με το συντελεστή του αγνώστου ώστε να απομονώσουμε τη μεταβλητή στο 1\textsuperscript{ο} μέλος και να βρεθεί η λύση.
\end{enumerate}
Αν η εξίσωση περιέχει :
\begin{itemize}[itemsep=0mm]
\item παρενθέσεις, τότε εκτελούμε τους πολλαπλασιασμούς ώστε να απαλοιφθούν οι παρενθέσεις και συνεχίζουμε ακολουθώντας την παραπάνω διαδικασία.
\item κλάσματα, τότε κάνουμε απαλοιφή παρονομαστών τα βήματα της οποίας είναι τα παρακάτω :
\begin{enumerate}[itemsep=0mm,label=\roman*.]
\item Υπολογίζουμε το Ε.Κ.Π. των παρονομαστών.
\item Πολλαπλασιάζουμε κάθε όρο της εξίσωσης με το Ε.Κ.Π.
\item Διαιρούμε το Ε.Κ.Π. με κάθε παρονομαστή όπου αυτός υπάρχει.
\end{enumerate}
Αν μετά από αυτά τα βήματα η εξίσωση περιέχει παρενθέσεις τότε συνεχίζουμε τη διαδικασία όπως περιγράψαμε προηγουμένως.
\end{itemize}
\Methodologia{Επαλήθευση} Αν γνωρίζουμε η υπολογίσουμε τη λύση μιας εξίσωσης, προειμένου να εξετάσουμε αν αυτή είναι η σωστή, τότε κάνουμε αντικατάσταση τον αριθμό αυτό στην αρχική εξίσωση και ύστερα από πράξεις εξετάζουμε αν η σχέση που θα προκύψει αληθεύει ή οχι.
